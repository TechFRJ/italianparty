# Italian Party Site

Site estatico modernizado para a Italian Party.

## Rodar localmente

```bash
npm start
```

Abra `http://localhost:3000`.

## Verificar

```bash
npm run check
```

## Deploy na Vercel

```bash
npm run deploy
```

Para deploy automatico, conecte este repositorio no painel da Vercel. Como o projeto e estatico, nao precisa de build command.
